# kfiatek-xyz

A personal portfolio and blog website built with Next.js, showcasing projects, work experience, and blog posts.

## Technologies Used

- **Next.js**: React framework for server-side rendering and static site generation.
- **TypeScript**: Typed JavaScript for better development experience.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **ESLint**: Linting for code quality.
- **PostCSS**: CSS processing tool.

## Project Structure

- `app/`: Next.js app directory with pages and layouts.
- `components/`: Reusable React components.
- `data/`: Static data for posts, projects, and work experiences.
- `public/`: Static assets like favicon and robots.txt.
